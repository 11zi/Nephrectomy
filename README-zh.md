# 启动

以开发模式编译并热重载

```sh
npm run dev
```

在 [Vitest](https://vitest.dev/) 运行单元测试

```sh
npm run test:unit
```

用 [ESLint](https://eslint.org/) 获得优化提示

```sh
npm run lint
```

# Nephrectomy
名为Nephrectomy的聊天室网页应用，旨在鼓励用户聚集聊天，拉近用户间的心理距离。
该项目也是iirose（蔷薇花园）聊天室项目的替代，网络上能搜索到这一项目。

1.  基础功能
    - 用户身份信息验证
    - 聊天
    - 聊天历史记录，并自动在数据库中做分页保存
    - 用户状态
    - 经济系统  
      在线可以获得固定经济
      - 聚集起来的人可以共同获得额外经济
      - 头目获得的更多
      - 头目可以调整所有人的分红（甚至倒贴）
      - 聊天会增加房间质量，使所有人收入增加
      - 每个房间向头目收固定费用作租金，防止房间过多
2.  自设图床  
    身份未验证的图片三天后删除  
    发送图片需要花费经济
3.  从蔷薇花园继承的功能
    - 小游戏，50%胜率
    - 转账功能
    - 用户主页信息展示
      称号
