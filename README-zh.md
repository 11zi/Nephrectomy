# Nephrectomy-肾脏切除

这个模板应当帮助您使用vue3的vite进行开发

## 自定义

See [Vite Configuration Reference](https://vite.dev/config/).

## 启动

### 以开发模式编译并热重载

```sh
npm run dev
```

### 在 [Vitest](https://vitest.dev/) 运行单元测试

```sh
npm run test:unit
```

### 用 [ESLint](https://eslint.org/) 获得优化提示

```sh
npm run lint
```

## 运营目标

1.  鼓励大家聚集
    基础功能

    - 用户身份信息验证
    - 用户状态
    - 聊天
    - 聊天历史记录保存
      对于每份自动分页的历史记录生成一个guid
    - 花里胡哨的功能负责增加互动

      - 炒股，背包，朋友圈，点歌，商店，骰子，签名，通话，状态

    - 经济系统
      在线可以获得固定工资
      - 聚集起来的人可以共同获得奖励
      - 头目获得的更多
      - 头目可以调整所有人的分红（倒贴）
      - 聊天会增加质量，使所有人收入增加
      - 每个房间收固定费用并收租金，防止房间过多
      - 多夫多妻制，更多的亲密关系收取更多的金

2.  鼓励大家分裂对立
    同房间内，每个话题会不断分裂产生分歧
    不同房间之间，玩家各自对立（自然关系）
    房间内可以创建其他房间，多的房间会多窗口排布在其他地方，这些窗口是临时的，话题结束后就会关闭
3.  否定当前花园的申必形态
4.  结昏离昏
    鼓励大家找对象，鼓励慢热
5.  自设图床
    未验证身份的图片三天后删除
    延迟两秒显示，防止白嫖
    发送大图需要花费金
6.  从蔷薇花园继承的功能
    - 赌博小游戏，50%胜率，盈亏自负
    - 转账功能
    - 用户主页信息展示
      称号
