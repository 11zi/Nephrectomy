# 启动

以开发模式编译并热重载

```sh
npm run dev
```

在 [Vitest](https://vitest.dev/) 运行单元测试

```sh
npm run test:unit
```

用 [ESLint](https://eslint.org/) 获得优化提示

```sh
npm run lint
```

# Nephrectomy-肾脏切除

1.  鼓励大家聚集
    基础功能
    - 用户身份信息验证
    - 用户状态
    - 聊天
    - 聊天历史记录保存
      对于每份自动分页的历史记录生成一个guid
    - 花里胡哨的功能负责增加互动
      - 炒股，背包，朋友圈，点歌，商店，骰子，签名，通话，状态

      在线可以获得固定工资
      - 聚集起来的人可以共同获得奖励
      - 头目获得的更多
      - 头目可以调整所有人的分红（倒贴）
      - 聊天会增加质量，使所有人收入增加
      - 每个房间收固定费用并收租金，防止房间过多
      - 更多的亲密关系收取更多的金

    发送大图需要花费金
